陆奇博士前沿创新趋势分享活动纪要

活动概述

  • 活动名称: 陆奇博士前沿创新趋势分享
  • 日期: 2025年6月25日
  • **地点：**浙江大学管理学院金色大厅
  • 核心主讲人: 陆奇博士 (奇绩创坛创始人)
  • 主要听众: 浙江大学的师生及创业者
  • 核心主题: 深入剖析由人工智能(AI)驱动的新时代格局与创业机会，重点阐述了三大机遇：研究型创业、智能体商业化、以及从中国出发建立全球化公司。

陆奇博士主旨演讲核心内容摘要

创业是技术与价值的核心纽带！

第一部分：新时代的格局 - 技术驱动创业

  • 时代定义: 当前时代由技术驱动创新，尤其是创业 。自21世纪以来，技术创业已成为创造财富最大的职业选择 。
  • 智能时代: 2022年11月ChatGPT的突破开启了智能时代 。
    • 生产工具: 模型、具身机器人。
    • 生产对象: 数据、算法、算力。
  • 生产力变革: 核心变革在于生产力从以“人”的能动性为主，转向以“硅基”的能动性、协同性和进化性为主。
  • 创业的价值:
    • 创业公司是利用技术创造价值的核心纽带，今天的科技巨头都曾是创业公司。
    • 技术驱动的创业具有强大的复利特性，可以随着时间实现指数级增长，因为技术的每个维度优化都可以带来相乘效应。

第二部分：新趋势 - 理解通用智能 (AGI)

  • 碳基与硅基的并行性: 人类（碳基）的通用智能是“大脑+交互界面（五官、手脚）”在环境中进化形成的。同理，硅基智能（AGI）也是“模型+交互界面”在数据和算力构成的环境中进化而来。
  • AGI价值创造的两大关键:
    1. 推理能力 (Reasoning): 找到完成任务的有效路径。
    2. 能动性 (Agency): 创造商业价值的核心。它指智能体为了达成目标，进行规划、使用工具并与环境交互操作的能力。陆奇博士用爱因斯坦（强推理）和奥本海默（强能动性）的例子来区分这两个概念。
  • 大模型的三步进化论:
    1. 第一步 (学知识): 通过预训练学习互联网的压缩知识。
    2. 第二步 (学思考): 通过数学和代码等进行思维链学习，提升思维能力。
    3. 第三步 (学专家技能): 在真实环境中，通过使用工具与环境交互来学习，不再依赖人类标注数据。当前阶段的核心是提升能动性和多轮对话中的意图理解与指令遵循能力。

第三部分：时代的三大机遇

  1. 机遇一：研究型创业公司 (Research-Founded) 成为主流
    • 范式变革: AGI让小团队也能利用算力探索科学前沿，使研究与价值创造的距离被前所未有地拉近。
    • 高斜率团队: 这类公司的成功关键是“高斜率”，即在认知、能力和价值上保持极快的成长速度。
    • 人人皆可研究: 借助AI工具，每个有独特认知和品味的人都有机会成为研究者，创办研究驱动的公司。
  2. 机遇二：2025，智能体 (Agent) 元年
    • Software 3.0: 软件开发正从“人写代码 (1.0)”和“数据训练模型 (2.0)”演进到人人都能“用自然语言写软件 (3.0)” 。
    • 大模型即操作系统: 大模型具备操作系统的核心功能：管理资源（上下文）、调度进程（Agent）、和提供编程环境（自然语言）。
    • 上下文是一切: 创业成功的关键在于找到并守住独特的“上下文”（数据环境），尤其是在企业、工厂等互联网之外的场景。中国因其高人口和产业密度，在上下文方面拥有独特优势。
  3. 机遇三：在中国做全球化公司
    • 供应链优势: AI时代的供应链是模型、数据、软硬结合的综合体系。中国在硬件、能源乃至模型（如DeepSeek、通义千问）的供应链上已具备全球竞争力。
    • 结构性机会: 中国的创业者如今可以像硅谷一样，从第一天起就以全球市场为目标进行创业。

第四部分：AI时代的四大商业赛道

  1. 认知智能（Cognitive AI）： 专注于模拟人类的认知能力，如理解、学习和推理。
  2. 场景智能（Contextual AI）： 强调AI在特定场景下的应用和优化。
  3. 具身智能（Embodied AI）： 将AI与物理世界相结合，使AI系统能够与环境进行交互。
  4. 科学智能（AI4S）： AI在科学研究领域的应用，加速科学发现。

第五部分：如何抓住机会 - 创业方法论

  1. 找准问题: 创业的起点不是找点子，而是找问题（找钉子）。要通过一路追问，找到一个自己充满热情、必须要解决的问题。

  2. 胆子要大: 价值等于“需求 x 稀缺性” 。稀缺性不仅来自技术攻坚，更来自勇气——做别人不敢做的事。

    Just Jump!

  3. 惠人达己，守正出奇: 惠利他人，适配个人兴趣热爱，一定要顺应时代大势（守正），同时找到自己独特的切入点和价值（出奇）。

  4. Build something weirder： 鼓励创新和差异化。

  5. 加入社区: 创业艰难，加入像奇绩一样的创业社区可以获得系统的支持和同伴。

  6. 满足下列三点之一：

    • 成为领域专家： 深入了解所在领域。

    • 动手做： 实践是最好的学习方式。

    • 加入一个创业公司： 从实践中学习创业经验。

奇绩创坛及创业者互动

  • 奇绩创坛: 陆奇介绍了奇绩的创业营、路演日、校友社区和投资理念（210万人民币换7%股权，无对赌不干预）。同时，奇绩也推出了创业公开课、算力支持和“前沿信号体系”等新服务。
  • 现场Office Hour:
    • AI SEO项目: 陆奇建议其探索按结果付费的商业模式，并点明该赛道天花板可能不高。
    • 科研AI合伙人项目: 在用户获取和定价上，陆奇建议做市场细分，并采取高定价策略来验证价值。
    • 开源生态Agent项目: 陆奇对此想法给予极高评价，反复催促创始人“赶快做”，并鼓励其立即申请奇绩创业营。
  • 校友分享: 多位浙大籍奇绩校友分享了创业经历，覆盖低空经济、AI Agent、制造业、出海等领域，并肯定了奇绩社区提供的巨大支持。

结论

本次奇绩论坛为浙大师生系统性地描绘了AGI时代从技术变革到商业机会的完整图景，并提供了清晰的创业路线图和方法论。活动强调了

认知、速度、勇气和社区在当前创业环境下的核心重要性，并鼓励在场听众抓住时代机遇，共创未来。

另附我印象较深的几点

  1. 结果收费：
    强调以效果为导向的收费

    • 从“按使用付费”到“按结果付费”： 陆奇博士认为，在当前的AI时代，商业模式正在演进 。过去的SaaS模式（软件即服务）让用户“用多少付多少”，而未来更好的模式是“把你结果做好以后，按结果付费” 。他强调，“结果收费会是更好，更容易买” 。

    • 特别适合中国市场： 他特别分析了为什么这种模式在中国更有效。与有长期企业软件付费习惯的美国市场不同，中国的老板更看重实际效果 。直接销售软件或服务功能，对方不一定买单，但如果能承诺并达成具体业务结果，交易就更容易达成 。

    • 具体案例： 在指导一个做“AI时代SEO”的创业项目时，他建议的收费模式不是卖软件，而是直接承诺结果。例如，帮助客户的品牌在AI问答中平均排到前三名，如果达不到就不收费。

  2. 高科技产品的高定价策略：

    • 定价要高： 陆奇博士建议，高科技产品的定价总体应该定得高一点 。
    • 筛选真实用户： 如果产品定价过低，吸引来的可能并非真正的核心用户，他们可能只是试用一下便会离开 。而愿意为高价付费的用户，通常是产品能切实解决其痛点的真实用户，他们的反馈更有价值 。
    • 验证产品价值： 如果在高价下仍有用户愿意购买，这本身就证明了产品具有高价值 。
    • 便于后期推广： 定价高，后续推广时永远可以打折；但如果一开始定价过低，后续再提价则非常困难 。

  3. 第二批用户与市场分割:

    • 识别“尝鲜者”： 创业者需要识别第一批用户是否为“尝鲜者” (Early Adopters) 。这类用户的特点是，无论产品好坏他们都愿意尝试，因此从他们身上获得的反馈不一定能完全代表市场需求 。
    • 市场分割寻找核心用户： 在“尝鲜者”之后，创业者必须进行“市场分割” (Market Segmentation)，去寻找下一波核心用户，即那些希望用新技术来追求梦想的“高瞻远瞩者” (Visionaries) 。
    • 补充知识而非认知： 陆奇博士指出，如何进行有效的市场分割，这不是一个认知问题，而是一个需要系统学习的知识问题 。
    • 推荐书籍： 他特别推荐创业者阅读《跨越鸿沟》 (Crossing the Chasm) 这本书来学习相关的方法论 。
